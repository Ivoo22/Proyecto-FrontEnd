FIT TRADE

FIT TRADE es una página web desarrollada como proyecto Frontend orientada a la venta y promoción de productos para pilates y entrenamiento funcional.

Funcionalidades principales:

- Página de inicio con novedades y productos destacados
- Sección de reseñas
- Formulario de contacto
- Navegación interna
- Footer con información de contacto y redes sociales
- Diseño Responsive
- Implementación de Grid y Flexbox
- Media Queries para distintos tamaños de pantalla

Estructura del sitio:

- Inicio
- Productos
- Reseñas
- Contacto

Proyecto publicado en Netlify: https://fittrade-projectofrontend.netlify.app/
